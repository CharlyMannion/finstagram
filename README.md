# FINSTAGRAM

A clone of Instagram, built with Ruby on Rails

## Getting started:

* `git clone https://github.com/CharlyMannion/finstagram`
* `bundle install`
*

## Info:

* Ruby 2.5.1p57

* Rails 6.0.2.2

* Starting up the Web Server:
```
rails server
```
